# Bootstrap Landing Page

A responsive **Landing Page** built using **Bootstrap** and modern front-end technologies.
This project demonstrates how to build a clean and responsive educational landing page for web development courses.

The website introduces beginners to different web technologies such as **HTML, CSS, JavaScript, PHP, and SQL**, and presents them in an organized and attractive layout.

---

# 🛠 Technologies Used

### Frontend

* HTML5
* CSS3
* Bootstrap 4

### Libraries

* Bootstrap Components
* jQuery
* Popper.js

---

# 🎯 Project Overview

This project is a **Landing Page for a Web Development Learning Platform**.

The page is designed to present course content, introduce programming tracks, and allow users to explore different technologies used in modern web development.

The design focuses on:

* Clean layout
* Responsive design
* User-friendly interface
* Organized content sections

---

# ✨ Features

### 📌 Responsive Navigation Bar

* Built with **Bootstrap Navbar**
* Includes course categories
* Dropdown for different development tracks

---

### 🧑‍💻 Hero Section

A landing hero section that introduces the platform and encourages users to start learning web development.

Includes:

* Headline
* Description
* Call To Action button

---

### 📚 Courses Section

Displays course cards for different technologies including:

* HTML
* CSS
* JavaScript

Each course card includes:

* Course image
* Course description
* Action button

---

### 📸 Gallery Section

An interactive **Bootstrap Carousel** that displays images related to web technologies such as:

* PHP
* SQL
* APIs

---

### 📩 Contact Section

A contact form allowing users to send messages.

Includes:

* Email input
* Message textarea
* Submit button
* Supporting image layout

---

### 🦶 Footer Section

Footer with:

* About section
* Social media links
* Copyright information

---

# 📱 Responsive Design

The layout adapts to different screen sizes using **Bootstrap Grid System**.

Supported devices:

* Desktop
* Tablet
* Mobile

---

# 🎨 UI Enhancements

Custom CSS was added to improve the UI:

* Hover animations for cards
* Styled carousel controls
* Section title design
* Custom footer styling
* Image optimization with `object-fit`
